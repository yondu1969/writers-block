Getting Started

Welcome to Mazaa React template! We are thankful to you for choosing our template for your project.

Now say goodbye to old techniques of making your website which are sluggish in loading, weird in looks and difficult to customize. Mazaa has all what you need to ensure a speedy online presence. This is a super fast template which loads with the blink of an eye and it will help you increase your conversion rate and ultimately The Sales. Mazaa can be used with its dashboard and it can be utilized for a restaurant, online food ordering and niches like this. Let start by providing valuable information to you.

Requirements

There are system requirements in order to install and setup mazaa template and its components properly. Make sure that you are running the node.js install npm install.

Recommended node and npm

                                        
                                            node install
                                            npm install

Install Npm Packages

NPM : You must have to npm install packages. You can do this by running npm install from the root of your project to install all the necessary dependencies.

                                    
                                       npm install.

Development server

Run npm run start for a dev server. Navigate to http://localhost:3000/ . The app will automatically reload if you change any of the source files. 

                                    
                                       npm run start.

Build

Run npm run build to build the project. The build artifacts will be stored in the build/ directory. Use the npm run build --prod flag for a production build.

                                    
                                       npm run build
                                       npm run build --prod
